# PercentProgressBar
自定义百分比圆形Progress
# SetUp 
### 1) In your root build.gradle:
allprojects {

    repositories {
        jcenter()
        maven { url 'https://jitpack.io' }
    }
}
### 2) In your library/build.gradle add:
<pre><code>
 compile 'com.github.icuihai:PercentProgressBar:v1.0'
</code></pre>
